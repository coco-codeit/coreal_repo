<h1 align="center">같이달램</h1>

## 프로젝트 소개

**같이 달램** 서비스는 저마다의 사람들과의 모임을 연결해주는 웹 서비스입니다.</br>
같이 달램에서 모임을 검색하고, 리뷰를 보며 마음에 드는 모임에 참여해보세요.

<br/>
<br/>

## 주요 기능

- 모임 생성/참여/취소, 모임 찜하기(로컬스토리지 이용)
- 모임 리뷰 작성
- 프로필 수정(프로필 이미지, 회사)
- 반응형 웹 서비스(PC, 태블릿, 모바일)

<br/>
<br/>
